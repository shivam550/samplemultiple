Your starting point to web mastery!

In this course you would learn how to design a website and code it to make it viewable on the web, on desktop computers, on mobile phones and tablets.

It will also serve as a strong base should you decide to further pursue a career in web development, web design or android application development.

HTML is the language of the web. CSS is the aesthetics that gives design to the page. jQuery is what makes a web page come alive and interactive. Learn all of this in one course plus a lot more.

  • Introduction to HTML/CSS & jQuery
  • Foundation HTML
  • Foundation CSS
  • HTML Inputs & Forms
  • Using DIVS & Best Practices
  • Web Page Segmentation
  • Positioning Page Elements
  • Photoshop Orientation
  • Converting Designs to HTML/CSS
  • Responsive Design
  • Introduction to jQuery
  • Basic interaction with jQuery
  • Deploying Your Site Live on the Web
